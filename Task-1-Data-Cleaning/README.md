# 📖 Project Overview

The objective of this task was to inspect, clean, validate, and prepare a raw dataset for analysis using Microsoft Excel. The dataset was thoroughly reviewed to ensure data quality, consistency, and accuracy before any analytical work.

Rather than making unnecessary modifications, the dataset was carefully examined, and only the required improvements were applied while preserving the integrity of the original data.

---

# 🎯 Objective

- Perform data quality assessment.
- Identify and resolve inconsistencies.
- Validate data accuracy.
- Prepare the dataset for further analysis.

---

# 🛠️ Tools Used

- Microsoft Excel

---

# ✅ Data Quality Checks Performed

The following data cleaning and validation steps were carried out:

- Inspected the complete dataset for overall data quality.
- Verified duplicate records across all columns; **no duplicate records were found.**
- Reviewed missing values and replaced blank **CouponCode** entries with **"No Coupon"** where appropriate.
- Checked text fields for formatting consistency.
- Verified numerical columns for accuracy and consistency.
- Reviewed date values to ensure valid formatting.
- Confirmed that the dataset contained consistent and complete records.
- Applied filters to all columns for easier exploration and analysis.
- Organized the worksheet to improve readability and usability.

# 📊 Outcome

The dataset was successfully reviewed, validated, and prepared for analysis. All required data quality checks were completed successfully. No duplicate records were identified, and the only necessary modification involved replacing blank values in the **CouponCode** column with **"No Coupon"** to improve data completeness while maintaining business meaning.

The final dataset is clean, organized, and ready for further data analysis and visualization.

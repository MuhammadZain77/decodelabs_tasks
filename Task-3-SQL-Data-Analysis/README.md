# 🗄️ Task 3 – SQL Data Analysis

## 📌 Project Overview

This project was completed as **Task 3** of the **DecodeLabs Data Analytics Internship Program**.

The objective of this task was to analyze a retail sales dataset using **MySQL** by writing SQL queries to extract meaningful business insights. The project demonstrates database creation, data exploration, filtering, sorting, aggregation, grouping, and business-oriented analysis.

---

# 🎯 Objectives

- Create and manage a relational database.
- Import and analyze a retail sales dataset.
- Perform SQL-based data exploration.
- Apply filtering and sorting techniques.
- Use aggregate functions for statistical analysis.
- Generate business insights using SQL queries.

# 📊 Task 2 - Exploratory Data Analysis (EDA) & Interactive Sales Dashboard

## 📌 Project Overview

This project was completed as **Task 2** of the **DecodeLabs Data Analytics Internship Program**.

The objective of this task was to perform **Exploratory Data Analysis (EDA)** on a retail sales dataset using **Microsoft Excel**, derive meaningful business insights, and design an interactive analytics dashboard for decision-making.

---

# 🎯 Objectives

- Explore and understand the retail sales dataset.
- Perform statistical analysis.
- Create meaningful KPIs.
- Analyze business performance using Pivot Tables.
- Build an interactive dashboard with Pivot Charts and Slicers.
- Present insights in a professional and visually appealing format.

---

# 📂 Dataset Information

The dataset contains retail sales transactions including:

- Order ID
- Order Date
- Customer ID
- Product
- Quantity
- Unit Price
- Payment Method
- Order Status
- Referral Source
- Coupon Code
- Total Price

---

# 📈 Exploratory Data Analysis (EDA)

The following analyses were performed:

- Total Orders
- Total Revenue
- Total Customers
- Total Products

- Product-wise Sales Analysis
- Revenue Analysis
- Order Status Analysis
- Payment Method Analysis
- Referral Source Analysis
